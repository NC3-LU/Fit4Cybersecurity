{% block report_intro_description %} {% blocktranslate %} As a building block of the CORAL EU-project, Fit4CSA is a questionnaire-based tool that helps assess the cybersecurity maturity of an ICT service, ICT process, an ICT web application or an ICT generic product.

The questions within Fit4CSA are separated into four streams depending on the target of assessment. Each stream of questions is grounded on existing standards and solid state-of-the-art best practices, as follows:

  • For ICT services, the relevant Fit4CSA questions are based on the candidate Cloud Services scheme (EUCS) (primarily its objectives and controls at the basic-level) issued by ENISA.
  • ICT processes, the relevant Fit4CSA questions are based on the standard ISO/IEC 21827:2008 Information technology — Security techniques — Systems Security Engineering — Capability Maturity Model.
  • For ICT web applications, the relevant Fit4CSA questions are based on OWASP requirements.
  • For ICT generic products, the relevant Fit4CSA questions are based on the Common Criteria based European candidate cybersecurity certification (EUCC) scheme issued by ENISA.
    • The answers are scored and depending on the scoring achieved, the tool proposes a set of recommendations. These recommendations can be implemented as first steps in order to improve the cybersecurity maturity of the service, process, web application or generic product for which the assessment was done. The answers chosen, the score and the recommendations are all included in the present report, which can be used as a conformity self-assessment in the spirit of the Cybersecurity Act.
