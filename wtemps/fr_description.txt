Description

Cet outil d'auto-évaluation est nommé Fitcyber4Africa pour souligner l'introduction à la sécurité de l'information. Il est aussi un prérequis pour faire un Diagnostic ACRC: un score minimal de 65/100 est requis. Le but pour cette restriction nous donne la possibilité pour mieux adapter les recommandations à votre organisation lors du Diagnostic.

Durant cette auto-evaluation $$qnumber$$ questions à choix multiples vous ont été posées. Après avoir analysé vos réponses, des recommandations vous ont été fournies.

Finalement, vous pouvez télécharger ce document en français, anglais ou en allemand ce qui vous aidera pour communiquer les résultats à vos fournisseurs si besoin.
